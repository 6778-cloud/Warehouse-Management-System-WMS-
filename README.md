# 📦 Warehouse Management System (WMS)

## 🧑‍💻 Overview
This project is a Web-based Warehouse Management System developed using PHP and MySQL.  
It supports inventory tracking, inbound/outbound management, and dashboard analytics.

---

## 🚀 Features
- Product Management (CRUD)
- Inventory Tracking System
- Inbound / Outbound Orders
- Dashboard with Analytics (API + JSON)
- User Authentication System

---

## 🛠️ Tech Stack
- Front-end: HTML, CSS, JavaScript
- Back-end: PHP
- Database: MySQL
- API: RESTful API (JSON)
